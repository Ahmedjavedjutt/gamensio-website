# Gamensio — Game Development Studio Website

**Developer:** Ahmed Javed
**Type:** Multi-page responsive website
**Tech Stack:** HTML5 · CSS3 · Vanilla JavaScript
**Pages:** Home · About Us · Portfolio/Services · Sign In

---

## Overview

Gamensio is a fictional game development studio website built as a
front-end development project. It showcases a professional multi-page
site with modern UI patterns including animated counters, live news
tickers, glassmorphism cards, and responsive navigation.

---

| Page | File | Description |

| Home | `index.html` | Hero, stats, featured games, news section |
| About Us | `htmlfiles/about-us.html` | Studio story, team, mission |
| Portfolio | `htmlfiles/services.html` | Game showcase and services |
| Sign In | `htmlfiles/signin.html` | User authentication UI |

---

## Features

- Live news ticker scrolling latest studio updates
- Animated statistics counter (5M+ players, 120+ games, 90+ countries)
- Hero section with floating Unity & Unreal Engine badges
- Glassmorphism card design (glass-card effect)
- Responsive hamburger navigation for mobile
- Scroll reveal animations on content sections
- Multi-page routing with consistent header and footer
- Google Fonts — Outfit + Laila typefaces
- Gradient background and gradient text effects
- Platform badges — Mobile, PC, PS4, VR
